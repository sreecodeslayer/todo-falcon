# To-do web App using Falcon
This is just a learning process using the basic stuffs for the most part

## How to Run?
1. Clone the repo and `cd` into the root dir
2. Set up a virtualenv `$ virtualenv ve`
3. Activate the virtualenv `$ . ve/bin/activate`
4. Install the modules and requirements `$ pip install -r requirements.txt`
5. Now run the server `$ gunicorn app:api`

To use the api alone, read the commented section inside the python files.
